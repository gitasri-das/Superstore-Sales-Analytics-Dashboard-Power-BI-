# 📊 Superstore Sales Analytics Dashboard (Power BI)
## Project Overview

Developed an interactive Power BI dashboard to analyze Superstore sales performance across regions, customer segments, product categories, states, and managers. The dashboard helps identify sales trends, profitability, and business performance through dynamic KPIs and visualizations.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Dashboard Features

## 📌 Executive Sales Overview
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Sales by Region
- Profit by Region
- Monthly Sales Trend
- Profit Margin by Region
## 📌 Order & Customer Analysis
- Top 10 Cities by Sales
- Sales by Order Priority
- Sales by Product Category
- Sales by Ship Mode
- Sales by Customer Segment
## 📌 Regional & Manager Performance
- Top 10 States by Profit
- Sales by State (Map)
- Sales by Manager
- Profit by Manager
## Key Features
- Interactive slicers (Region, Ship Mode, Order Priority, Manager)
- Multi-page dashboard navigation
- Custom region sorting using DAX
- Month sorting for chronological trend analysis
- KPI cards with dynamic filtering
- Interactive cross-filtering across visuals
## Business Insights
- Identified the highest-performing regions and states.
- Compared regional sales and profitability.
- Analyzed customer purchasing patterns.
- Evaluated manager-wise sales and profit performance.
- Tracked monthly sales trends for better business decisions.
